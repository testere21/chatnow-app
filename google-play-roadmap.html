1
Google Play Console Hesabı
DETAYLI ADIMLAR:
1. play.google.com/console adresine git

SONUÇ: Developer hesabın aktif olur
Bekliyor
2
Release Keystore Oluştur
DETAYLI ADIMLAR:
1. Bilgisayarında "cmd" aç (Windows tuşu + R, cmd yaz)
2. Proje klasörüne git: cd C:\Users\PC\Desktop\yeniFerhat\MyTodoApp3\android
3. Keystore oluştur: keytool -genkey -v -keystore release-key.keystore -alias release-key -keyalg RSA -keysize 2048 -validity 10000
4. Şifre belirle (UNUTMA! Bu şifreyi kaydet)
5. Kişisel bilgilerini gir (ad, şirket, şehir, ülke)
6. Keystore dosyası oluşur: release-key.keystore
7. Bu dosyayı GÜVENLİ yere yedekle (USB, bulut)
ÖNEMLİ: Bu dosyayı kaybedersen uygulamayı güncelleyemezsin!
Bekliyor

🔧 FAZE 2: TEKNİK
4
App Bundle (AAB) Oluştur
DETAYLI ADIMLAR:
1. Proje klasöründe android/app/build.gradle dosyasını aç
2. Release keystore ayarlarını ekle:
signingConfigs {
  release {
    storeFile file('release-key.keystore')
    storePassword 'ŞİFREN'
    keyAlias 'release-key'
    keyPassword 'ŞİFREN'
  }
}
3. buildTypes → release kısmında: signingConfig signingConfigs.release
4. CMD aç: cd android
5. AAB oluştur: gradlew bundleRelease
6. Dosya hazır: android/app/build/outputs/bundle/release/app-release.aab
SONUÇ: Play Store için AAB dosyası hazır
